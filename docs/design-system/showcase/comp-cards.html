마커 탭 시 지도 위에 표시되는 프리뷰 카드. 280-320px. 진입 애니메이션 포함.

사용 가이드라인
✅ Do
  • 마커 위에 위치시키되 화면 상단 근접 시 아래로 표시
  • 가격(Bold) + 점수(배지) + 주소(1줄) 정보만 간결하게 표시
❌ Don't
  • 카드 안에 상세 정보(카테고리별 점수 등)를 넣지 않음
  • 지도 마커와 프리뷰 간 거리가 너무 멀어지지 않도록 주의
접근성
  • 마커 클릭 시 프리뷰 출현을 aria-live="polite"로 알림
  • 키보드 탭으로 마커 순회 가능, Enter로 프리뷰 오픈

CardSelector §3.2

수평 칩 배열. 선택: brand-500 bg + 흰 텍스트. 미선택: outline.

사양
속성토큰값비고
선택 배경--color-brand-500#0891B2 + 흰 텍스트
미선택 스타일—outline (1px border)
정렬 옵션—종합 점수순 / 예산 적합도순 / 통근 시간순기본: 종합 점수순
폰트—12px
패딩—6px 16px
사용 가이드라인
✅ Do
  • 결과 리스트 상단에 수평 배열로 배치
  • 기본 정렬을 "종합 점수순"으로 설정
❌ Don't
  • 정렬 옵션을 5개 이상으로 확장하지 않음
  • 칩 레이블에 "추천순" 사용 금지 (§5 컴플라이언스)
접근성
  • role="radiogroup" + 각 칩에 role="radio" + aria-checked
  • 키보드 화살표로 칩 간 이동 가능
