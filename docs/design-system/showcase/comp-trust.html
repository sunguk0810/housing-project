ExternalLinkCTA §3.6

외부 링크 버튼 + 클릭 시 면책 모달. "바로 문의" / "바로 연결" 대신 "외부 페이지로 이동".

Pattern A: 인라인 단일 링크
외부 사이트로 이동합니다
Pattern B: 복수 플랫폼
각 외부 사이트로 이동합니다. 매물 정보는 해당 플랫폼에서 확인해주세요.

컴플라이언스 노트
§5.4 외부 링크 CTA 규칙. 면책 모달은 필수이며, "본 서비스는 정보 분석 플랫폼이며 부동산 중개 서비스가 아닙니다" 문구를 포함해야 합니다. CTA에 "매물 문의", "상담 신청", "중개사 연결" 표현을 사용할 수 없습니다.

외부 사이트로 이동

네이버 부동산으로 이동합니다.

매물 정보는 해당 외부 사이트에서 제공하며, 본 서비스는 정보 분석 플랫폼이며 부동산 중개 서비스가 아닙니다.

매물 정보의 정확성은 해당 외부 사이트에서 확인해주세요.
