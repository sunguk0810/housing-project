사용 가이드라인
✅ Do
  • CTA 버튼은 Accent(코랄) 색상으로 시각적 강조
  • 전체 너비 CTA는 하단 Sticky로 배치 (모바일 최적화)
  • Disabled 상태에서 이유를 텍스트로 안내 ("필수 항목을 입력해주세요")
❌ Don't
  • 한 화면에 Primary + Accent CTA를 동시 배치하지 않음 (시각적 혼란)
  • Disabled 버튼을 이유 안내 없이 표시하지 않음
  • "추천" 또는 "문의" 같은 금지 문구를 버튼 텍스트에 사용하지 않음 (§5 컴플라이언스)
접근성
  • button 시맨틱 태그 사용 (div/span 금지)
  • Disabled 상태에서 aria-disabled="true" + 사유 텍스트 연결 (aria-describedby)
  • 최소 44×44px 터치 타겟 확보 (WCAG 2.5.5)
  • 포커스 링: outline: 2px solid + outline-offset: 2px
사용 페이지
전체 페이지 (공통 UI 패턴)
