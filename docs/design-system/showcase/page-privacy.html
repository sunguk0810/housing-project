Privacy (/privacy)

§4.3 — 개인정보처리방침.

개인정보처리방침
개인정보처리방침
제1조 (처리 목적)

집콕신혼은 아래 목적으로 개인정보를 처리합니다. 처리 목적 이외의 용도로는 사용하지 않으며, 변경 시 별도의 동의를 받습니다.

  • 주거 조건 분석을 위한 임시 데이터 처리
  • 서비스 이용 통계 (비식별 처리)
제2조 (처리 항목)

서버에 영구 저장하는 개인정보는 없습니다

소득, 자산, 직장 주소 등은 세션 내에서만 처리되며 분석 완료 후 즉시 삭제됩니다.

제3조 (보유 기간)
항목 보유 기간
분석 입력 데이터 세션 종료 시 즉시 삭제
카카오 계정 식별자 회원 탈퇴 시 즉시 삭제
서비스 이용 기록 비식별 처리 후 통계 목적
제4조 (파기 절차)

보유 기간이 경과하거나 처리 목적이 달성된 개인정보는 지체 없이 파기합니다. 전자적 파일은 복구 불가능한 방법으로 영구 삭제합니다.
