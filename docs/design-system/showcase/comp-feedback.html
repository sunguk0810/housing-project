Tooltip §3.7

ⓘ 아이콘 탭 → 다크 배경 툴팁. 3단계 계층적 공개 데모.

기본 툴팁
예산 적합도 ⓘ
소득, 자산, 대출 여력을 기반으로 해당 단지의 가격이 예산 범위 내에 있는지 평가합니다.
3단계 계층적 공개
1단계: ⓘ 탭 1줄 요약
소득 대비 매매가·전세가의 부담 수준을 평가합니다
2단계: 더보기 요인 기여도
PIR(가격 대비 소득) 40% + DTI(부채 상환 비율) 30% + 보유 현금 충분도 30%
가중치 합산으로 0-100 환산
3단계: 전체 방법론 상세 문서 링크
분석 방법론 전체 문서 → 별도 페이지 이동
분석 방법론 자세히 보기 →
사양
속성토큰값비고
트리거—ⓘ 아이콘 탭
배경--color-neutral-900#1C1917 (웜 다크)
텍스트—#FFFFFF
코너--radius-md8px
3단계 공개—ⓘ→1줄 요약→요인 기여도→전체 방법론계층적 디스클로저
사용 가이드라인
Do
  • "왜 이 정보가 필요한지" 1줄 해설을 제공
  • 3단계 계층적 공개 패턴으로 점진적 정보 제공
Don't
  • 툴팁에 3줄 이상의 텍스트를 넣지 않음 (1단계)
  • hover 트리거 사용 금지 (모바일에서 hover 불가 → 탭 트리거만 사용)
접근성
  • ⓘ 아이콘에 aria-label="정보 보기" + aria-expanded 토글
  • 툴팁에 role="tooltip" + id 참조
  • 트리거와 툴팁을 aria-describedby 로 연결

WhyTooltip §3.7+

"왜 이 정보가 필요한가요?" 신뢰 구축 목적의 인라인 툴팁. 기존 Tooltip과 구분.

부부 합산 연봉
소득 정보는 PIR(가격소득비)와 DTI(부채상환비율) 계산에 사용됩니다. 예산 적합도 점수의 핵심 요인입니다. 세션 종료 시 즉시 삭제됩니다.
사양
속성토큰값비고
트리거—"? 왜 필요한가요?" 텍스트 버튼pill 형태
배경--color-neutral-100호버: neutral-200
팝업—기존 Tooltip과 동일 스타일다크 배경 + 흰 텍스트
내용—정보 필요 이유 + 삭제 시점 안내신뢰 구축 목적
사용 가이드라인
✅ Do
  • 민감 정보(소득, 자산) 입력 라벨 옆에 배치
  • 삭제 시점을 함께 안내하여 신뢰 강화
❌ Don't
  • 모든 입력 필드에 남용하지 않음 (민감 정보만)
  • ⓘ 아이콘 대신 "? 왜 필요한가요?" 텍스트로 더 명시적인 트리거 사용
접근성
  • aria-expanded 토글로 팝업 상태 전달
  • 트리거에 aria-label="왜 합산 연봉 정보가 필요한지 안내 보기"
