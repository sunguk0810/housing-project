AmountInput §3.4

금액 입력 필드. tabular-nums, 쉼표 구분. 빠른 입력 버튼: +1,000만 / +5,000만 / +1억.

  • 실시간 값 변경 시 aria-live="polite" 로 알림

ConsentForm §3.6

동의 체크박스. 필수 미체크 시 CTA 비활성. 세션 종료 시 즉시 삭제.

CTA 비활성 상태 (필수 미체크)
사양
속성토큰값비고
[필수] 개인정보—직장 권역, 예산 범위세션 종료 시 즉시 삭제
[선택] 정밀 분석—보유 현금, 상세 소득세션 종료 시 즉시 삭제
[필수] 위치정보—직장 주소→좌표 변환좌표 미저장

사용 가이드라인
✅ Do
  • 필수 미체크 시 CTA를 비활성(disabled)으로 유지
  • 선택 동의 없이도 기본 분석이 가능함을 안내
  • 보유 기간 "세션 종료 시 즉시 삭제"를 명확히 표시
❌ Don't
  • 선택 동의를 필수인 것처럼 유도하지 않음
  • 별도 페이지 이동 없이 인라인 동의 수집
접근성
  • 각 체크박스에 aria-required (필수 항목)
  • 그룹에 role="group" + aria-label="동의 항목"
