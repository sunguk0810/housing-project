# Step 2 Interview Guide

Status: blocked (Human Validation unavailable)  
Date: 2026-02-13  
Target: 맞벌이 신혼부부 (결혼 예정~결혼 2년)

## 0) Mode

- 현재 모드: `AI Proxy only`
- 제약: 실제 인터뷰/리쿠르팅 실행 불가
- 운영 방침: 인터뷰 가이드는 보존하되, 당장은 `AI Red-Team`으로 대체 검증

Method note:

- 현재까지의 인터뷰 결과는 타겟 퍼소나를 대입한 LLM 기반 가상 응답이다.
- 실제 사용자 인터뷰가 가능한 시점에는 아래 절차로 전환해야 한다.
  1. 타겟 리쿠르팅(신혼/맞벌이/탐색 시점 기준)
  2. 현재 행동 흐름 확인(앱/엑셀/중개사 사용 순서)
  3. 고통 지점과 의사결정 기준 꼬리질문
  4. 프로토타입 과제 수행 관찰(입력 이탈/문의 트리거)
  5. 인터뷰 결과와 정량지표를 함께 판정

## 1) Interview Goal (재개 시)

- 집 탐색에서 반복되는 고통 지점 확인
- `재무 + 2직장 통근 + 보육 + 치안` 통합추천 가치 검증
- 민감정보 입력 장벽 정량 확인

## 2) Screener (재개 시)

- [ ] 서울/경기/인천 거주 또는 거주 예정
- [ ] 결혼 예정 또는 결혼 2년 이내
- [ ] 최근 6개월 내 집 탐색 경험
- [ ] 맞벌이(또는 맞벌이 예정)

## 3) Core Questions (12, 재개 시)

1. 집 탐색 시작 계기와 마감 시점은?
2. 전세/매매 우선순위와 이유는?
3. 어떤 앱/채널을 어떤 순서로 쓰는가?
4. 탐색 과정에서 가장 피곤한 작업은?
5. 통근 절대 기준(예: 60분)과 타협 기준은?
6. 통근 계산에서 민감한 요소(환승/도보/혼잡)는?
7. 보육 필수 조건은?
8. 치안 판단 기준은?
9. 예산은 어떻게 계산하는가?
10. 연봉/대출/직장주소 입력 의향(0~10)은?
11. 어떤 결과가 행동(문의)을 가장 유도하는가?
12. 어떤 조건에서 문의/방문으로 이어지는가?

## 4) Follow-up Prompts (꼬리질문)

- 통근 60분은 계산 기준인가, 체감 기준인가?
- 입력을 멈춘 정확한 필드는 무엇인가?
- 어떤 표현에서 "광고 같다"고 느끼는가?
- 이 서비스가 필요 없다면 이유는 무엇인가?
- 문의를 망설이게 하는 마지막 리스크는 무엇인가?

## 5) Scoring Guide

Pain Score (1~5):

- 1: 불편 거의 없음
- 3: 번거롭지만 감수 가능
- 5: 즉시 해결이 필요한 수준

Sensitivity Score (0~10):

- 0~3: 입력 강한 거부
- 4~6: 조건부 수용
- 7~10: 적극 수용

Action Intent (0~100%):

- 0~30: 탐색만
- 31~70: 조건부 문의
- 71~100: 즉시 문의
