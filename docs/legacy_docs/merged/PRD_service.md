# 제품 요구사항 문서(PRD) – 신혼부부 주거 추천 서비스

## 1. 개요

이 문서는 1인 개발자가 구현할 **신혼부부 주거 추천 서비스**의 제품 요구사항을 정의합니다. 목표는 예산, 통근, 보육, 학군, 안전 인프라 데이터를 통합한 맞춤형 아파트 추천 서비스를 제공하되, 부동산 중개가 아닌 정보 분석 플랫폼으로 포지셔닝하는 것입니다. 기존 플랫폼들이 예산 역산, 보육·치안 데이터 통합, 2인 통근 고려 등에서 미흡한 점을 보완하는 것을 핵심 가치로 삼습니다【103981203960773†L11-L16】【103981203960773†L41-L53】.

## 2. 목적 및 목표

- **목적**: 신혼부부(특히 맞벌이)의 주거 선택 과정에서 정보 부족 문제를 해결하고, 예산·통근·보육·학군·치안 요소를 종합하여 최적의 단지를 추천한다.
- **목표**: 사용자에게 상위 10개 단지의 종합 점수와 이유를 제시하며, 입력 데이터는 서버에 저장하지 않는다【543942488311355†L15-L21】.
- **성공 지표**: 사용자의 추천 결과 만족도 설문(예: 4점 이상/5점), API 호출 성공률 99%, 페이지 로딩 시간 < 2초, 개인 정보 저장률 0%.

## 3. 사용자 페르소나

1. **맞벌이 신혼부부** – 두 직장의 통근 시간을 동시에 고려해야 하고, 첫 주택 구입 또는 전세를 고민 중이다. 어린이집·학군·안전도에 민감하며 예산 한도가 명확함.
2. **미혼 혹은 1인가구** – 포트폴리오 데모를 체험하기 위해 사용하는 잠재적 투자자·개발자.

## 6. 데이터 및 외부 연동

- **국토교통부 실거래가 API** – 아파트 매매/전세 거래 이력. 2025년부터 호출 제한이 확대되어 하루 2,000건까지 가능.
- **어린이집·유치원 통합 API** – 위치, 정원, 평가등급 등 정보를 제공. 필요 시 정원 대비 재원율을 계산해 지표화.
- **학교알리미 API** – 학교 위치, 학업 성취도 지표 제공.
- **행정안전부 재난안전데이터 API** – 2025년 12월 31일 기준 CCTV 설치 현황 및 대피시설 정보를 REST API로 제공【784864165518395†L74-L91】.
- **카카오 지도 API** – 주소 검색, 지도 표시 및 좌표 변환.

## 7. 사용자 흐름

1. **방문 및 입력** – 사용자가 홈페이지에 접속하여 거주 계획 입력 폼을 작성한다.
2. **예산 및 통근 계산** – 서버는 입력값을 받아 가용 예산과 통근시간을 계산한다.
3. **후보 추출** – 예산과 기본 필터를 만족하는 단지를 DB에서 추출한다.
4. **지표 수집 및 점수화** – 각 단지의 어린이집·학군·안전 인프라 지표를 계산하고 가중치를 적용해 종합 점수를 산출한다.
5. **결과 표시** – 지도와 카드 리스트로 상위 10개 단지를 보여주고 외부 링크를 제공한다.
6. **정책 안내** – 사용자가 정책 정보 페이지에서 주거지원 제도와 개인정보 처리방침을 확인할 수 있다.
