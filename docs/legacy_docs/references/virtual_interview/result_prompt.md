역할:
너는 리서치 adjudicator다.
입력으로 GPT/Gemini/Claude의 "가상 리서치 결과" 3개를 받는다.
목표는 상충 내용을 정리하고 Step2 실행용 단일 결론을 만든다.

요청:
1) 3개 결과의 공통점/충돌점/누락점 정리
2) 충돌 항목은 "보수적 가정" 기준으로 단일안 채택
3) Step2 확정안 작성:
- 최종 pain point Top 5
- 최종 KPI 가정치(보수/기준/낙관)
- 최종 Go/No-Go 판정 조건
- 인터뷰 대체 리서치의 한계와 보완 계획
4) 바로 실행 가능한 TODO 10개를 우선순위로 작성

출력 형식:
A. Consensus Summary
B. Disagreement Log
C. Final Step2 Decision Pack
D. TODO Top 10 (우선순위 순)
E. Risk Register (5개)

규칙:
- 과장 금지
- "가상 리서치 기반"임을 명시
- 실제 사용자 검증 필요 항목 별도 표시
- 한국어로 작성
