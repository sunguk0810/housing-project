3) 실제 탐색 플로우(앱/포털/중개사/지인) 단계별로 “가장 피곤한 작업”은? [가정]

6) 보육(어린이집/유치원)에서 “도보 10분” 외에 필수(대기/연장/차량/평가)는? [가정]

### 랜딩/설문 실험안 2개(단계별)
**실험 1: 랜딩페이지(개인화 미리보기) + 대기자 모집**  
- Step1: 카피 2종 A/B: (A) “2직장 60분 내 + 보육10분 + 예산으로 단지 Top10” vs (B) “가능지역을 한 번에(전세/매매 탭)”. [가정]  
- Step2: 최소 입력(현금·연봉·직장1/2·월상한) → “가능 동네 예시 3개”만 보여주고, 전체 Top10은 이메일/카톡으로 받기. [가정]  
- Step3: KPI: (i) 완료율, (ii) 대기등록률, (iii) “매물 문의 클릭(가상 CTA)” 클릭률. 최종 KPI를 **문의 클릭률**로 정렬(핵심지표 일치). [가정]  

**실험 2: 설문(정량) + 컨시어지 추천(세미수동)**
- Step1: 설문에서 가중치(예산/통근/보육/치안/학군) 100점 배분 + 데이터 입력 민감도(연봉/대출/주소). [가정]  
- Step2: 상위 30명에 한해 “24시간 내 추천 리포트 PDF/노션” 제공(단지 10개+외부 매물 링크+월고정비 추정). [가정]  
- Step3: KPI: (i) 리포트 열람률, (ii) 문의 클릭률(외부 링크 클릭), (iii) 추천 만족도/NPS, (iv) “유료 의향(월 구독/1회 리포트)” 비율. [가정]  

### 수요 검증 임계치(Go/No-Go)
- **Go**(Step2 종료 기준): (1) 인터뷰 20명 중 14명 이상이 “현재 프로세스가 매우 고통(4/5 이상)” + (2) 50% 이상이 “연봉/대출/직장 주소 입력 가능(조건부 포함)” + (3) 실험1에서 가상 문의 클릭률 ≥ 12% 또는 실험2 컨시어지 리포트 문의 클릭률 ≥ 20%. [가정]  
- **No-Go/피벗**: 클릭은 높으나 “재무 입력 거부”가 높으면 → “월상한 기반(민감도↓) 추천”으로 피벗. [가정]  
- 임계치 설정 이유: 초기 MVP의 사업성은 “추천 정확도”보다 **행동(문의 클릭)**으로 검증하는 편이 빠르기 때문. [가정]

## F. 리스크 Top5+대응
1) **매물(호수) 데이터 확보 실패** → 공공 실거래 전월세는 동/호 미제공이라 대체 불가. 대응: P0는 “대표 매물 수기/제휴 + 외부 링크”, P1에서만 자동동기화 계약. citeturn19view1

3) **치안 점수의 낙인/편향** → 통계는 집계단위·주기 한계, 관할 경계 데이터 부재. 대응: “치안은 참고지표” 고지, 세부지표 투명 공개(근거 링크), 사용자 OFF 토글. citeturn19view5turn28view1
