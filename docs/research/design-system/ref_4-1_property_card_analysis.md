| **2** | **거래유형+가격 "매매 8억 5,000"** | ~24-26px **Bold**, 검정, **"억" 단위 + 천 단위 쉼표** |
| **3** | 86C/59m² · 101동 31층 | ~14px Regular, **dot(·) 구분**, 회색 |
| **4** | 고층, 뻥뷰, 2년전 올수리 | ~13px Regular, **쉼표 구분**, 회색 — 매물 특징 태그 |
| 우측 | **3D 평면도 이미지** | 약 120×100px, 라운딩 ~8px, 우측 배치 |

**스타일**
- 패딩: 약 16-20px
- 라운딩: 보이지 않음 (카드 전체 캡처가 아닐 수 있음)
- 그림자: 미세하거나 없음
- 보더: 하단에 얇은 구분선 추정
- 배경: 순백 #FFFFFF

**특이점** ⭐ 한국 가격 표기 패턴
- **"매매"** 거래유형이 가격과 같은 줄에 같은 크기 → 거래유형이 가격의 일부
- **"8억 5,000"**: "억" 한자어 단위 + 천 단위만 쉼표 → 한국 부동산 표준 표기
- "실매물 확인" 배지: 오렌지/코랄 텍스트로 신뢰 장치 역할
- **단지명이 가격보다 위(1순위)**: 한국 서비스는 "어디"가 먼저, 미국은 "얼마"가 먼저
- 이미지가 우측 소형 (3D 평면도) → 사진이 아닌 도면으로 정보 가치 극대화
- 면적 표기: **86C/59m²** (공급/전용 병기) — 한국 특유 패턴

## 3. 집콕신혼 적용 인사이트

### ✅ 채택 권장

**네이버 부동산에서 (가장 유사한 구조):**
- **이미지 없는 텍스트 카드의 정보 배치**: 단지명 1순위 → 가격 → 스펙 순서가 우리와 거의 동일
- **배지 시스템**: 상단에 상태 배지("집주인"), 하단에 날짜 배지("확인매물 26.02.14.") → 우리는 상단에 순위 배지, 가격 옆에 출처 태그
- **면적 표기 패턴**: 145C/102m² → 한국 사용자에게 익숙한 "공급/전용" 병기, 우리도 동일 패턴 사용
- **전체 여백감**: 16-20px 패딩이 텍스트 전용 카드에서 충분한 공기감 제공 → 우리 p-4(16px) 적절 확인

**직방에서:**
- **한국 가격 표기법**: "매매 8억 5,000" — **거래유형 + 억 단위 + 천 단위 쉼표**
  → 우리도 "매매 3억 2,000만" 형식 채택 (design_brief 확인 완료)
- **dot(·) 구분자**: 한국 서비스 표준 → 우리도 dot 채택
- **"실매물 확인" 오렌지 텍스트**: 신뢰 장치를 텍스트 컬러로 표현 → 우리의 출처 태그 컬러 참고

**Redfin에서:**
- **넉넉한 여백**: 정보 간 6-8px 간격이 가독성 최고 → 우리 카드도 정보 행 간 최소 6-8px 유지
- **액션 아이콘 위치**: 가격과 같은 행 우측에 공유+찜 → 우리는 단지명 행 우측에 원형 게이지, 하단에 "비교 추가"
- **정보 성격별 구분자 차별화**: 핵심 스펙은 공백, 부가 정보는 dot → 우리도 고려 가능

**Zillow에서:**
- **이미지 위 배지 오버레이**: 반투명 검정 pill 패턴 → 우리는 이미지 없으나, 향후 지도 마커 위 배지에 응용
- **중개사 정보 최소화**: ALL CAPS + 10px + 극연한 회색 → 법적 필수 정보를 시각적으로 최하위 배치하는 패턴

### 💡 신규 인사이트

1. **한국 가격 표기 "억/만" 단위에서의 tabular-nums 적용 범위**
    - 직방: "8억 5,000" — 숫자만 고정폭, "억" 글자는 가변폭
    - 네이버: "20억" — 단순
    - → 우리 구현 시: `font-variant-numeric: tabular-nums`는 숫자에만 적용되므로
      "3억 2,000만"에서 숫자 부분 자동 고정폭, "억"/"만" 한글은 별도 처리 불필요

2. **카드 내 정보량 — 우리가 가장 많다**
    - Zillow: 5행 (가격/스펙/주소/중개사)
    - 네이버: 6행 (배지/단지명/가격/스펙/설명/중개사)
    - **우리: 7-8행** (배지+단지명 / 주소 / 세대수·준공·면적 / 실거래가 / 4점수 2×2 / 통근시간 + 비교)
    - → 카드 높이가 길어질 수 있음. **p-4(16px) 패딩은 유지하되, 행 간 간격을 4-6px로 타이트하게 조절** 필요
    - 또는 4점수 그리드를 compact 모드(6px 바)로 표시하여 높이 절약

3. **한국 서비스의 "이미지 없음" 카드 존재 확인**
    - 네이버 부동산이 이미지 없는 텍스트 카드를 실제로 사용 중
    - → 우리 카드가 이미지 없어도 한국 사용자에게 낯설지 않음 (선례 있음)

4. **출처/날짜 배지의 위치**
    - 네이버: 하단에 "확인매물 26.02.14." 오렌지 배지
    - → 우리도 카드 하단에 출처 태그 배치 고려: `[국토부 실거래 2026.01]`
    - 현재 스펙에서는 출처 태그 위치가 카드 내에 명시되지 않음 → **가격 옆 인라인 또는 카드 최하단** 중 결정 필요
