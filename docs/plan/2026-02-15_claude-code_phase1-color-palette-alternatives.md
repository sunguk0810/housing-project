# 리서치 결과

## 1. WCAG 대비율 매트릭스

### 1.1 계산 방법론

WCAG 2.1 기준 상대 휘도(Relative Luminance) 및 대비율(Contrast Ratio) 계산:

1. HEX -> RGB(0-255) 변환
2. 각 채널 0-1 정규화 (val/255)
3. sRGB -> 선형 변환: `val <= 0.04045 ? val/12.92 : ((val+0.055)/1.055)^2.4`
4. 상대 휘도: `L = 0.2126*R_lin + 0.7152*G_lin + 0.0722*B_lin`
5. 대비율: `(L_bright + 0.05) / (L_dark + 0.05)`

**WCAG AA 기준**: 본문 텍스트 4.5:1 이상 | 큰 텍스트(18pt+)/UI 컴포넌트 3.0:1 이상

### 1.2 배경색 상대 휘도

| 배경 | HEX | RGB | 상대 휘도 L |
|------|-----|-----|------------|
| 흰색 | #FFFFFF | (255,255,255) | 1.000000 |
| Stone-50 | #FAFAF9 | (250,250,249) | 0.955348 |
| 웜 다크 | #1C1917 | (28,25,23) | 0.010040 |
| 다크모드 | #0F172A | (15,23,42) | 0.008815 |

### 1.3 Primary 컬러 대비율 매트릭스

| 대안A Teal-600 | #0D9488 | 0.2304 | **3.74:1** (aa\*) | **3.59:1** (aa\*) | **4.67:1** (AA) | **4.77:1** (AA) |
| 대안A Emerald-600 | #059669 | 0.2286 | **3.77:1** (aa\*) | **3.61:1** (aa\*) | **4.64:1** (AA) | **4.74:1** (AA) |
| 대안B Blue-500 | #3B82F6 | 0.2355 | **3.68:1** (aa\*) | **3.52:1** (aa\*) | **4.75:1** (AA) | **4.85:1** (AA) |

> **범례**: AA = 본문 텍스트 4.5:1 충족 | aa\* = 큰 텍스트/UI 3.0:1만 충족 | FAIL = 3.0:1 미달

**핵심 발견**:
- 모든 후보가 **라이트 배경(흰색/Stone-50)에서 본문 텍스트 AA 4.5:1을 미달**한다
- Indigo-500만 라이트 배경에서 4.47:1로 AA에 가장 근접하나, 다크 배경에서는 오히려 대비가 떨어진다
- 현재 #0891B2와 Blue-500은 휘도가 거의 동일(0.235)하여 대비율 특성이 동일하다
- **라이트 배경 본문 텍스트에 Primary를 직접 사용하려면, 모든 후보에서 700-800 단계의 어두운 변형이 필요**하다

### 2.2 대안별 상세 분석

#### 대안A: Deep Teal/Emerald 계열

**장점**:
- 라이트 배경 대비율이 현재안 대비 미세 개선 (3.74-3.77 vs 3.68)
- 프롭테크 시장에서 독보적 포지셔닝 (경쟁사 중 녹색 계열 서비스 드묾)
- 자연/생태 연상으로 "주거+환경" 가치 전달에 유리
- Emerald-600(#059669)은 Tailwind 기본 팔레트에 포함되어 토큰 생성 용이

**단점**:
- H=161-175 범위는 "금융 신뢰감"보다 "건강/환경" 이미지가 강함
- 적색맹(Protanopia) 사용자에게 녹색 계열이 황갈색으로 보여 의도된 브랜드 인상 왜곡 가능
- Coral Orange(#F97316) 대비 보색 각도가 줄어들어 시각적 긴장감 감소
- 기존 디자인 시스템 전면 교체 필요 (9단계 스케일 재정의)

#### 대안B: Slate Blue/Indigo 계열

**장점**:
- Blue-500(#3B82F6)은 대중적으로 검증된 "신뢰" 색상
- Indigo-500(#6366F1)은 라이트 배경에서 4.47:1로 AA에 가장 근접
- 금융/핀테크 사용자 기대에 부합하는 색상

**단점**:
- **치명적 차별화 실패**: 직방(#326CF9), 토스(#0064FF), Zillow(#006AFF)와 동일 색상 범주
- Blue-500의 밝기(L=60%)가 지나치게 높아 UI 요소에서 "가벼운" 인상
- Indigo-500은 다크 배경에서 대비율이 3.92-4.00:1로 AA 미달
- "프롭테크 블루 피로도"에 기여할 위험

#### CTA 버튼 색상과 전환율 관련 연구

웹 리서치 결과, CTA 버튼 색상과 전환율에 대한 주요 연구 결과:

- **주변 대비가 핵심**: 절대적으로 "최고" 색상은 없으며, 배경/주변 요소와의 대비가 가장 중요 (출처: [CXL - Which CTA Button Color Converts the Best?](https://cxl.com/blog/which-color-converts-the-best/), 기준일: 2024)
- **오렌지/레드 계열의 긴급성**: HubSpot A/B 테스트에서 빨강이 녹색 대비 21% 전환율 증가, Dmix에서 검정->노랑 변경 시 34% 증가 (출처: [OptinMonster](https://optinmonster.com/which-color-button-converts-best/), 기준일: 2024)
- **블루 vs 오렌지**: 한 테스트에서 블루가 오렌지 대비 9% 우위, 다른 테스트에서는 반대 결과 -> 컨텍스트 의존적 (출처: [CXL](https://cxl.com/blog/which-color-converts-the-best/), 기준일: 2024)
- **핵심 원칙**: "최적 CTA 색상 = 주변과 가장 높은 대비를 만드는 색상" (출처: [Amra and Elma - Buy Now Button Statistics 2025](https://www.amraandelma.com/buy-now-button-color-test-statistics/), 기준일: 2025)

**결론**: 현재 Teal 배경에서 Coral Orange(#F97316)는 보색 대비(167도)를 제공하여 CTA 주목성에 가장 유리하다.

### 6.2 구체적 개선 권장 사항

#### 우선순위 1 (필수): Score 등급 휘도 교정

- Good: #42A5F5 -> #2196F3 또는 더 어두운 블루
- Average: #90A4AE -> #B0BEC5 또는 더 밝은 회색
- 목표: 인접 등급 대비율 최소 1.5:1 이상

#### 우선순위 2 (권장): Primary 텍스트용 어두운 변형 확보

- 라이트 배경 본문 텍스트에 #0891B2를 직접 사용하면 AA 미달(3.68:1)
- 텍스트용으로 700단계 이상의 어두운 변형(예: #0E7490, #155E75) 사용 필요
- 현재 9단계 스케일의 700(#0E7490) 사용 시 대비율 확인 필요

#### 우선순위 3 (선택): Score 등급에 보조 식별 수단 추가

- 색상만으로 등급을 구분하지 않도록 패턴/아이콘/라벨 병행
- WCAG 1.4.1 "색상으로만 정보 전달 금지" 원칙 준수
