# Plan Execute: 부동산 프로필별 스코어링 시뮬레이션

## 목표

5개 가중치 프로필(균형형/예산중심/통근중심/대단지중심/가치극대화)에 따른 스코어링 결과 차이를 가상 후보 단지 기반으로 시뮬레이션하고, 프로필별 순위·점수·강점/약점 변화를 비교 분석한다.

## 범위

- In Scope:
  - 다양한 특성의 가상 아파트 후보 12개 생성
  - 5개 가중치 프로필별 정규화 점수 + 최종 점수 산출
  - 프로필별 Top 5 순위 변화 비교
  - 가치극대화 프로필의 다양성 재정렬(diversity rerank) 효과 확인
  - 시뮬레이션 스크립트 `scripts/simulate-profiles.ts` 작성
- Out of Scope:
  - 실제 DB 데이터 조회 (가상 데이터 사용)
  - 예산 계산(budget.ts) 연동 (maxPrice 직접 지정)
  - API 엔드포인트 수정
- 참조 SoT: PHASE1 S4 (스코어링 로직, 가중치 프로필)

## 작업 단계

1. 가상 후보 단지 12개 데이터 설계 (다양한 가격대/통근시간/보육/치안/학군/규모 조합)
2. `scripts/simulate-profiles.ts` 작성 — 엔진 함수 직접 호출
3. 시뮬레이션 실행 및 결과 출력 (콘솔 테이블)
4. 결과 분석 및 Plan 문서 기록

## 검증 기준

1. `tsx scripts/simulate-profiles.ts` 실행 시 에러 없이 결과 출력
2. 5개 프로필 모두에 대해 12개 후보의 점수·순위가 정상 산출
3. 프로필별 Top 5 순위가 의미 있는 차이를 보임 (같은 순위가 아님)
4. 가치극대화 프로필에서 diversity rerank 적용 결과 확인

## 결과/결정

- 상태: `done`
- 핵심 결과:
  - 12개 가상 후보 단지 × 5개 프로필 = 60회 스코어링 정상 산출
  - **프로필별 Top 1 차이**: 균형형/예산중심/대단지중심/가치극대화 → 래미안 강남포레스트(78.9~81.4점), 통근중심 → 힐스테이트 판교역(75.4점)
  - **순위 변동이 가장 큰 단지**: 브라이튼 여의도(±6위), 자이 위례신도시(±5위) — 프로필에 따라 순위가 크게 달라짐
  - **순위 고정 단지**: e편한세상 시흥(10위), 호반써밋 인천(11위), 한양수자인 김포(12위) — 모든 지표가 낮으면 프로필과 무관하게 하위권
  - **다양성 재정렬**: 가치극대화 프로필에서 premium 3개(30%)/mid 4개(40%)/value 3개(30%) 목표 분포를 정확히 달성. 더샵 광명역(value 티어)이 6위→4위로 상승
  - **프로필 효과 검증**: 통근 중심은 commute weight 38%로 판교역 1위, 대단지 중심은 complexScale 25%로 자이 위례신도시 2위로 상승 — 가중치 차별화가 의도대로 작동
