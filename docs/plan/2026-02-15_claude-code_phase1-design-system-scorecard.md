| 기준 | 가중치 | 측정 방법 |
|------|--------|----------|
| **접근성 (Accessibility)** | 35점 | WCAG AA 대비율 충족률, 색맹 시뮬레이션 통과, KRDS 준수 |
| **브랜드 적합도 (Brand Fit)** | 25점 | 감성 키워드(신뢰/따뜻함) 매칭, 경쟁사 차별화 거리 |

**의사결정 컷오프**:
- 현재안 >= 75점: **유지** (변경 불필요)
- 대안이 현재안보다 15점+ 상회: **변경 권장**
- 차이 < 15점: **부분 개선** 검토
